OpenClaw Windows 一键安装说明书

适用对象：不会自己装 Node.js、不会配 URL / Key、希望把 OpenClaw 和常用渠道一次配好的 Windows 用户。

本安装包覆盖模型提供商、渠道配置、插件补装、skills 选择安装、安装摘要和辅助启动脚本。

1. 这套安装包做了什么

  • 调用 OpenClaw 官方 Windows 安装脚本，自动处理 Node.js 和 OpenClaw CLI。
  • 在安装过程中按步骤提示填写模型 API Key、Base URL、模型 ID。
  • 可选配置 Feishu、QQ Bot、Telegram、Discord、Slack、LINE、WhatsApp。
  • 只有在用户勾选了某个渠道后，才会继续询问该渠道需要的 token、secret、webhook 路径或回调 URL。
  • 提供推荐 skills 选择安装，并支持手动输入额外 skill slug。
  • 安装后生成摘要文件和几个常用的 `.bat` 辅助脚本。

2. 使用方法

方式 A：交互式

  1. 双击 install-openclaw.bat。
  2. 按向导选择模型来源。
  3. 填写 API Key、Base URL、模型 ID 等信息。
  4. 勾选要接入的渠道。
  5. 为已勾选渠道填写凭据；暂时不配的可以跳过。
  6. 勾选要安装的 skills。

方式 B：预设配置

  1. 复制 installer-config.example.json。
  2. 改名为 installer-config.local.json。
  3. 预先填好模型、渠道、skills。
  4. 双击 install-openclaw.bat 直接安装。

3. 模型配置

模式需要填写说明
OpenAIapiKey, modelRef默认示例是 openai/gpt-5.4。
OpenRouterapiKey, modelRef默认示例是 openrouter/auto。
MoonshotapiKey, modelRef默认示例是 moonshot/kimi-k2.5。
自定义兼容 OpenAIbaseUrl, apiKey, providerId, customModelId适用于中转站、自建兼容接口或本地推理网关。

如果你的后端报 content must be string，把 requiresStringContent 改成 true。如果工具调用会出错，把 supportsTools 改成 false。

4. 渠道和要填的凭据

渠道要填什么备注
Feishu / LarkApp ID、App Secret；Webhook 模式额外要 Verification Token、Encrypt Key默认推荐 websocket，不需要公网回调地址。
QQ BotAppID、AppSecret安装器会按需补装 @openclaw/qqbot。
TelegramBotFather token首次私聊通常需要 pairing。
DiscordBot token要去 Discord Developer Portal 开 bot 并启用 intents。
SlackSocket 模式：bot token + app token；HTTP 模式：bot token + signing secretHTTP 模式还要在 Slack 平台填回调 URL。
LINEChannel access token、Channel secret需要在 LINE Developers 里填写 webhook 回调地址。
WhatsApp无固定 token；安装后走 QR 登录安装器可直接拉起 openclaw channels login --channel whatsapp。

5. 渠道回调 URL 怎么看

如果你选择了需要回调地址的模式，安装器会询问 Public gateway base URL。例如：

https://bot.example.com

随后安装摘要里会自动帮你拼出：

  • Slack HTTP: https://bot.example.com/slack/events
  • LINE: https://bot.example.com/line/events
  • Feishu webhook: https://bot.example.com/feishu/events

6. Skills 选择安装

安装器内置了一组推荐 skill 选项，用户可以多选，也可以额外手输 slug。

Slug作用
files文件查找、整理和清理
calendar-planner日程安排
pgPostgreSQL 相关
open快速打开文件和链接
docker-skillDocker 操作
skill-shell给第三方 skill 配安全 shell 能力
find-skills-for-clawhub辅助查找 ClawHub 上的 skill
openclaw-master-skills大而全的技能包，谨慎安装

7. 安装完成后会生成什么

  • ~/.openclaw/openclaw.json
  • ~/.openclaw/.env
  • 安装摘要：outputs/openclaw-install-summary.txt
  • 辅助脚本目录：launchers/

8. 常见问题

双击没反应

powershell -NoLogo -NoProfile -ExecutionPolicy Bypass -File .\install-openclaw.ps1

安装完找不到 openclaw

关掉当前终端，重新开 PowerShell，再试：

openclaw --version

Slack / LINE / Feishu webhook 没有收到事件

  • 先确认公网回调 URL 可访问。
  • 确认平台控制台里填的路径和安装摘要里生成的一致。
  • 再看 openclaw gateway status 和 openclaw logs --follow。

WhatsApp 一直没连上

重新执行：

openclaw channels login --channel whatsapp

9. 安全建议

不要把 .env、API Key、AppSecret、bot token 上传到公开仓库。打包发给别人时，仓库里只放示例配置，不放真实密钥。

10. 截至 2026-04-11 的官方依据

  • OpenClaw 官方 Windows 安装入口：install.ps1
  • 内置或打包渠道包含 Discord、Telegram、Slack、Feishu、LINE、QQ Bot、WhatsApp 等
  • openclaw skills install <slug> 用于从 ClawHub 安装 skill
